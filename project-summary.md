# Airbnb-Style Property Rental Platform

**Role:** Full Stack Developer  
**Technologies:** Node.js, Express.js, MongoDB, Mongoose, EJS, Tailwind CSS, Multer, REST APIs

## Key Features & Responsibilities:
- Developed full-stack property rental platform with user authentication and session management
- Implemented MVC architecture with 3 controllers, 3 routers, and data models for users/properties
- Created image upload system using Multer with custom filename generation and MIME type validation
- Integrated MongoDB Atlas cloud database with Mongoose ORM for data modeling
- Built responsive UI with Tailwind CSS and EJS templating engine
- Implemented route protection middleware for authenticated user access control
- Developed file management system with multiple static directories for asset serving
- Designed RESTful endpoints for property CRUD operations and booking management
- Configured automated CSS processing with Tailwind CLI watch mode
- Set up MongoDB session store for persistent user sessions

## Technical Highlights:
- Handled file uploads with custom storage configuration and validation
- Implemented session management with express-session and connect-mongodb-session
- Created utility functions for secure random string generation
- Set up error handling middleware for 404 page rendering
- Configured production-ready MongoDB Atlas connection with error handling
